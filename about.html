About QuizApp

Learn more about how QuizApp works and what it can do for you.

Overview

QuizApp is an interactive platform that allows you to create and participate in quizzes easily. Whether you're an educator, student, or trivia enthusiast, QuizApp offers a seamless way to challenge your knowledge and have fun!

Features

  • Create Quizzes: Design quizzes with multiple-choice questions and a timer. Share your quiz with friends using a unique code.
  • Participate in Quizzes: Enter the quiz code provided by the creator and test your knowledge against a variety of questions.
  • Track Results: Creators can view the results of participants, allowing for insightful feedback and learning.
  • User Accounts: Securely log in and manage your quizzes through your personalized account.

Technology Stack

Frontend: Our frontend is developed using modern web technologies and hosted on Vercel for fast and reliable performance.

Backend: The server-side is powered by Node.js and Express, with data management handled by MongoDB and Mongoose for robust storage solutions.

How to Use

Creating a Quiz

Log in, navigate to the create quiz page, fill in the quiz details, and share the generated code with others.

Joining a Quiz

Simply enter the quiz code on the homepage and start answering questions.
